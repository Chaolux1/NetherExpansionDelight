# Jaden's Nether Expansion Delight

## 📜 Essence

This mod adds 30+ unique food items and recipes, bringing together Jaden’s Nether Expansion and Farmer’s Delight in one seamless experience. It keeps the vanilla vibe but spices things up with Nether-themed cooking.

## 🎯 Features

- 30+ new food items based on Nether biomes, mobs, and resources.
- Full support for Farmer’s Delight: cutting, cooking, crafting.
- Balanced food stats that fit into vanilla survival.
- Exotic Nether ingredients: Glowcheese, Hogham Meat, Wisp Essence.
- From light snacks to big meals: dumplings, pies, burgers.
